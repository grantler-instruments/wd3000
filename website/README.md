# WD3000 marketing site

Vite + React + MUI + Zustand landing page using the Grantler dark theme.

```bash
npm install
npm run dev      # http://localhost:5174/wd3000/
npm run build    # dist/ for GitHub Pages (base /wd3000/)
```

Routes: `/` (home), `/privacy` (privacy policy — App Store URL).  
Deployed Pages also serve the web app at `/app/` (assembled by root `npm run build:pages`).

From the repo root: `npm run website:dev` / `npm run website:build`.
